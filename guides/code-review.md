# 코드 리뷰 가이드

## 봇 4명 자동 리뷰 체계

PR이 열리면 4명의 AI 봇이 자동으로 코드 리뷰를 수행합니다.

### 설정 파일

| 파일 | 역할 | 위치 |
|------|------|------|
| `.github/workflows/label-pr.yml` | Copilot 리뷰 요청 + 자동 라벨링 | 레포 루트 |
| `.github/labeler.yml` | 경로 기반 라벨 규칙 | 레포 루트 |
| `.coderabbit.yaml` | CodeRabbit 언어 설정 | 레포 루트 |

### 봇별 리뷰 포인트

**Gemini Code Assist**
- 전체적인 코드 품질 평가
- 로직 오류, 엣지 케이스 지적
- 리팩토링 제안

**CodeRabbit**
- 변경 사항 요약 (Walkthrough)
- 파일별 상세 리뷰
- Actionable comments with 코드 제안

**Copilot**
- 인라인 코드 제안
- 간결한 전체 리뷰

### 라벨 자동화

PR 본문에 "Generated by Claude Code"가 포함되면 자동으로 `ai-generated` + `claude-code` 라벨이 붙습니다. 경로 기반 라벨(`frontend`, `backend`, `docs`)도 자동 부착됩니다.
